## 🚀 Cara Menjalankan Project

1. Clone repository

```bash
git clone https://github.com/abrianhiskiasiregar/KumpulanTugas-Pemrograman-Aplikasi-Mobile-Abrian-Hiskia-Siregar.git
```

2. Masuk ke folder project yang ingin dijalankan

```bash
cd nama_project
```

3. Install dependency

```bash
flutter pub get
```

4. Jalankan aplikasi

```bash
flutter run
```
